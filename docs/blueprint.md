# **App Name**: EduVista SMP

## Core Features:

- Integrated PPDB Enrollment System: Full student registration pipeline with document upload capabilities and real-time status tracking for parents and administrators.
- Smart School Dashboard: A comprehensive CMS to manage news, school calendars, student achievements, and academic schedules with instant data persistence in Firestore.
- AI Content Optimizer tool: An AI-powered editor tool that helps administrators generate summaries and SEO-friendly tags for school news and event descriptions.
- Dynamic Academic Center: Interactive modules for displaying teaching staff directories, lesson schedules, and downloadable study materials for students.
- Media Vault & Gallery: Automated image gallery with lightbox features and categorized albums for documenting school activities and extracurricular programs.
- Real-time Notifications: Instant broadcast system for school-wide announcements and deadline reminders integrated into the user interface.
- Unified Admin Panel: A secure portal for multi-role user management, exportable registration data, and real-time analytics for site engagement.

## Style Guidelines:

- The primary color is a sophisticated Academic Navy (#1E3A8A) chosen for its association with trust and institutional wisdom. This is paired with an elegant Amber Gold accent (#F59E0B) for secondary elements to convey prestige and excellence.
- Backgrounds utilize a highly desaturated pearl blue tint (#F8FAFC) to maintain a crisp, clean aesthetic that prioritizes readability across all screen sizes.
- A font pairing of 'Poppins' for headlines to provide a contemporary, structured feel, and 'Inter' for body text for maximum clarity and cross-platform accessibility.
- Modern grid system using high-end Glassmorphism effects for headers and info cards, ensuring a 'floating' clean appearance with substantial white space.
- Subtle, weightless page transitions and delicate hover states on action buttons and cards to emphasize a high-speed, modern technical feel.
- Thin-stroke, minimal vector icons to represent different academic categories, keeping the visual cognitive load low for busy parents and students.
